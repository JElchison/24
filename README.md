# 24
Solver for Suntex game "Twenty Four"

## Example Usage

```
$ ./24.py 3 3 7 8
((3 *   3) +  7)  + 8   = 24
 (3 *   3) + (7   + 8)  = 24
((3 *   3) +  8)  + 7   = 24
 (3 *   3) + (8   + 7)  = 24
((3 *   3) +  7)  + 8   = 24
 (3 *   3) + (7   + 8)  = 24
((3 *   3) +  8)  + 7   = 24
 (3 *   3) + (8   + 7)  = 24
 (7 +  (3  *  3)) + 8   = 24
  7 + ((3  *  3)  + 8)  = 24
 (7 +  (3  *  3)) + 8   = 24
  7 + ((3  *  3)  + 8)  = 24
 (7 +   8) + (3   * 3)  = 24
  7 +  (8  + (3   * 3)) = 24
 (7 +   8) + (3   * 3)  = 24
  7 +  (8  + (3   * 3)) = 24
 (8 +  (3  *  3)) + 7   = 24
  8 + ((3  *  3)  + 7)  = 24
 (8 +  (3  *  3)) + 7   = 24
  8 + ((3  *  3)  + 7)  = 24
 (8 +   7) + (3   * 3)  = 24
  8 +  (7  + (3   * 3)) = 24
 (8 +   7) + (3   * 3)  = 24
  8 +  (7  + (3   * 3)) = 24
```
